# 프로젝트 이름

Seed

## 시작하기

오픈마켓 프로젝트는 사용자들이 상품을 등록하고 구매할 수 있는 플랫폼을 제공합니다.<br>
백엔드 부분은 자바와 스프링부트 프레임워크를 사용하였습니다.<br>
프론트엔드 부분은 HTML, CSS, JavaScrip를 통해 구현되었습니다

### 전제 조건

JDK 17<br>
Oracle DB 19C<br>
IntelliJ<br>
